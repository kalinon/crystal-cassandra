    class Cassandra::DBApi::Connection

    • Cassandra::DBApi::Connection
    • DB::Connection
    • Reference
    • Object

    Overview

    Represents a Cassandra connection. It contains cluster configuration and a session. The connection is also used to construct statements from strings.

    Defined in:
